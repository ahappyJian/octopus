octopus
=======

a type casting system on java
A simple language, contains four statements:
A:=string
A=(int|long|float|double|string)var
A=B op C
print A

Note: op=\*+-/.
Example 1:
a:=10.0
b:= 20;c=a+b;print c;

Example 2:
a:=10.0
b=version;c=b.a;printc;

Example 3:
a:=20
b=(double)a;
c=b/10;
print c;
d=a/10;
print d;
